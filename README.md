## Run Locally

Clone the project

```bash
  git clone https://github.com/Aniketsudke/Dry-Bean-Classification.git
```

Go to the project directory

```bash
  cd Dry-Bean-Classification
```

Install dependencies

```bash
  pip install flask numpy pandas scikit-learn
```

Start the server

```bash
  python main.py
```
